## üöÄ Getting Started

### Prerequisites

- Node.js (Latest LTS version recommended)
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/Uspectacle/PsycoQuizz.git

# Navigate to project directory
cd PsycoQuizz

# Install dependencies
npm install

# Start development server
npm run dev
```
